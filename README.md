## 📖 О проекте

> **Туризм Кыргызстана работает вслепую.** Отели, юрты, каршеринги и гиды работают через WhatsApp и Excel. У государства нет единой картины потоков и выручки. Информация разбросана по десяткам сайтов и соцсетей.

**Unigency AI** — это один продукт, который решает эти проблемы сразу для трёх аудиторий: туристов, турбизнеса и государства.

- **🤖 AI-маршруты** — персональные туры по запросу на русском языке
- **🗺 Интерактивная карта** — точки и дни тура визуализированы
- **📦 CRM для агентств** — туры, бронирования, аналитика в одном окне
- **📈 Сквозная аналитика** — от клика до брони
- **🔗 Интеграции** — Booking, Яндекс.Такси, Ticket.kg, Google сервисы
